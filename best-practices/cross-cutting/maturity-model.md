## Maturity Levels

| Level | Name | Description |
|:-----:|------|-------------|
| 0 | Absent | No process exists |
| 1 | Ad Hoc | Unstructured, reactive, person-dependent |
| 2 | Repeatable | Basic procedures exist but inconsistently followed |
| 3 | Defined | Documented, standardized, communicated |
| 4 | Managed | Measured, controlled, predictable |
| 5 | Optimized | Continually improved, automated where beneficial |
